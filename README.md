# 📈 Trade Analyzer Dashboard

A Flask-based analytics dashboard that helps traders analyze their trading performance using uploaded CSV trade data.

## Features

- Total Profit Analysis
- Win Rate Calculation
- Best & Worst Stock Identification
- Average Holding Time
- AI Trading Coach
- Behaviour Analysis
- Sector-wise Performance Analysis
- Strategy-wise Performance Analysis
- Trading Companion Chat
- Interactive Plotly Charts

## Technologies Used

- Python
- Flask
- Pandas
- Plotly
- HTML
- CSS
- Bootstrap

## How to Run

1. Clone the repository

```bash
git clone <repository-link>
```

2. Install dependencies

```bash
pip install -r requirements.txt
```

3. Run application

```bash
python app.py
```

4. Open

```text
http://127.0.0.1:5000
```
